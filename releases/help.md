# Справка по расширению

**Контакты:** digital-mind@yandex-team.ru | 2026

## Общая информация
- Данное расширение предназначено для автоматизации массовой рассылки заранее сформированного текстового сообщения предварительно определённому списку пользователей в Яндекс Мессенджере (в браузерной версии).
- Рассылка производится методом полной эмуляции действий пользователя производимыми в окне мессенджера при массовой рассылке сообщений. 
- Работа расширения базируется на взаимодействии с элементами интерфейса Яндекс Мессенджера посредством манипуляций с DOM элементами и использовании селекторов. Поскольку расширение опирается на текущую структуру веб‑страницы, любые существенные изменения со стороны разработчиков платформы (например, рефакторинг HTML‑разметки, смена классов/ID элементов, обновление JavaScript‑фреймворка) могут привести к нарушению его функциональности.
- К сожалению, разработчики не предоставляют API для управления мессенджером, поэтому, работа расширения жёстко завязана на содержимое веб страницы.
- Расширение не собирает и не хранит персональные данные.
- При обнаружении некорректной работы сообщите автору.
- Автор производит разработку и поддержку приложения исключительно на добровольных началах, но будет категорически не против, если руководство "Команды поддержки бизнеса" решит простимулировать разработку и дальнейшую техническую поддержку приложения.


## Авторизация

Авторизация пользователя носит номинальный характер. Её цели:
- Ограничить неконтролируемое распространение расширения.
- Сбор статистических данных о востребованности расширения.
Токен выдаётся на 180 дней (0.5 года), если в запросе не оговорён другой срок.
(срок взят по аналогии времени действия пароля на Стаффе)
